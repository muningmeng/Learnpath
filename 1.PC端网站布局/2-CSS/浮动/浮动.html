圣杯布局

前言

首先，我们需要知道啥是圣杯布局，再思考圣杯布局是如何实现的？今天我们就来说说 —— 圣杯布局的原理及实现的三种方法：浮动、flex、grid。

什么是圣杯布局？

圣杯布局是左右栏固定，中间栏自适应的网页布局。

它有以下这几点要求：

  • 头部（header）和尾部（footer）占据屏幕的所有宽度。
  • 中间栏左右定宽，中间部分自适应。
  • 中间部分的高度最高。

实现圣杯布局的方法

1.1 浮动


    <div class="header">header</div>
    <div class="container">
      <div class="center">center</div>
      <div class="left">left</div>
      <div class="right">right</div>   
    </div>
    <div class="footer">footer</div>
  

    * {
        margin: 0;
        padding: 0
    }

    body {
        min-height: 700px
    }

    .header,.footer {
        width: 100%;
        height: 50px;
        line-height: 50px;
        text-align: center;
        background: #666;
    }

    .center,.left,.right {
        position: relative;
        float: left;
        min-height: 130px;
        line-height: 130px;
        text-align: center;
    }

    .container {
        padding: 0 220px 0 200px;
        overflow: hidden;
    }

    .left {
        width: 200px;
        margin-left: -100%;
        left: -200px;
        background: #7bd;
    }

    .right {
        width: 220px;
        margin-left: -220px;
        right: -220px;
        background: #f63;
    }

    .center {
        width: 100%;
        background: #d6d6d6;
        word-break: break-all;
    }
    
    .footer {
        clear: both;
    }

1.2 flex


    <div class="header">header</div>
    <div class="container">
      <div class="left">left</div>
      <div class="center">center</div>
      <div class="right">right</div>   
    </div>
    <div class="footer">footer</div>
  

    * {
        margin: 0;
        padding: 0
    }

    body {
        min-height: 700px
    }

    .header,.footer{
       width:100%;
       height: 80px;
       line-height: 80px;
       text-align: center;
       background: #666; 
    }

    .container{
       display: flex;
       min-height: 100px;
    }

    .center{
        flex:1;
        background: #d6d6d6;
    }

    .left{
        width: 200px;
        background: #7bd;
    }

    .right{
        width: 220px;
        background: #f63;
    }

1.3 grid


   <div class="container">  
    <div class="header">header</div> 
    <div class="left">left</div>
    <div class="center">center</div>
    <div class="right">right</div>   
    <div class="footer">footer</div>
   </div>  
  

    * {
        margin: 0;
        padding: 0
    }

    body {
        min-height: 700px
    }

    .container{
        display: grid;
    }

    .header{
        grid-row:1;
        grid-column: 1/5;
        background: #666;
    }

    .left,.center,.right{
        grid-row:2;
    }

    .left{
        grid-column: 1/2;
        background: #7bd;
    }

    .center{
        grid-column:2/4;
        background: #d6d6d6;
    }

    .right{
        grid-column:4/5;
        background: #f63;
    }

    .footer{
        grid-row:3;
        grid-column: 1/5;
        background: #666;
    }

双飞翼布局

前言

双飞翼布局和圣杯布局的达到的效果基本相同，都是侧边两栏宽度固定，中间栏宽度自适应。 主要不同之处是解决中间被挡住的问题时，圣杯布局在父元素上设置了padding-left和padding-right，在给左右的内容设置了position:relative，通过左右移动得到了很好的展示； 而双飞翼布局是给中间部分新定义一个div，给新定义的div设置margin-left和margin-right。

实现双飞翼布局的方法


   <div class="header">header</div>
   <div class="container">
     <div class="center">
       <div class="center-inner">content</div>
     </div>
     <div class="left">left</div>
     <div class="right">right</div>   
   </div>
   <div class="footer">footer</div>
  

    * {
        margin: 0;
        padding: 0
    }

    body {
        min-height: 700px
    }

    .header,
    .footer {
        width: 100%;
        height: 80px;
        line-height: 80px;
        text-align: center;
        background: #666;
    }

    .left,
    .right,
    .center {
        float: left;
        min-height: 300px;
    }

    .center {
        width: 100%;
        background: #d6d6d6;
    }

    .center .center-inner {
        margin: 0 220px 0 200px;
    }

    .left {
        width: 200px;
        margin-left: -100%;
        background: #7bd;
    }

    .right {
        width: 220px;
        margin-left: -220px;
        background: #f63;
    }
    .footer{
        clear: both;
    }
